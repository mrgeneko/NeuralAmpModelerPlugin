07/20/2026 - v0.8.0 First Anti-Static release
  - Parametric models: a model's own knobs and switches appear in the plug-in,
    laid out to fit however many it declares, up to 12. Parameters with a fixed
    set of positions render as segmented switches instead of continuous knobs.
  - The knob row is always reserved rather than hidden behind a click, so the
    window is taller (550px) than upstream's.
  - A DAW's generic parameter list shows a loaded model's real parameter names
    instead of placeholders.
  - Knob value readouts are capped at two decimal places. Drag granularity is
    unchanged.
  - Renamed to Anti-Static. It installs as AntiStatic.vst3 with its own plug-in
    and manufacturer IDs, so it coexists with an existing Neural Amp Modeler
    install rather than replacing it.
  - Built against mrgeneko/NeuralAmpModelerCore, with link-time optimisation and
    fast-math enabled on both platforms.
  - No model is bundled. A demo parametric model is attached to the release,
    since ordinary .nam models have no parameters for the knob row to show.
  - Windows: the Third Party Notices link in the settings page works again; it
    was broken by the plug-in bundle being flattened during packaging.
