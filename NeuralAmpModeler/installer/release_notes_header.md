## Downloads

- **Windows** — `Anti-Static-__VERSION__-win.zip` (installer)
- **macOS** — `Anti-Static-__VERSION__-mac.dmg`
- **Demo model** — `muffinator_v4_optimal.param.nam`, for trying the parametric knobs (no model is bundled)

The `*-pdbs.zip` / `*-dSYMs.zip` are debug symbols, only useful for diagnosing crashes.

## Before you install

**macOS** builds are signed and notarized (Developer ID) — Gatekeeper should let the installer and app open with no warning.

**Windows** builds are not code-signed. SmartScreen warns on the installer and standalone the first time — choose **More info → Run anyway**. The VST3 loads in a DAW without any prompt.
